Titulo : Node Api

Descrição: Criação de uma Api para um site como o OLX.

Tecnologias:
     NodeJS, Javascript, MongoDB, ExpressJS.
     CORS: Usei o CORS, para poder usar os recursos da minha API de dominios diferentes.

Funcionalidades: 
    Criação, update, delete and read users.
    criação, update, delete and read products.
    Login e reset password.

Base de dados:
    Collection: Users 
    tables: username, email, password, phone number, location.
    Relação com a collection products.

    Collection: products
    tables: title, description, category, price.
    Relação com a tabela users e com categories.

Cronograma:
    read user/users
    user register/update/delete.
    authenticate user.
    forgot/reset password.
    read/product/products.
    product create/update/delete.
